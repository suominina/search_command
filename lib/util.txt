$ コマンド > ファイル //リダイレクト
	ls > file //lsの結果をfileに出力
$ コマンド >> ファイル//リダイレクト
	ls >> file //lsの結果をfileに追記
	ls 存在するファイル 存在しないファイル 2> file //エラーが出た時のみfileに出力
	ls 存在するファイル 存在しないファイル &> file //標準出力、標準エラー出力の両方をbothファイルに格納
	コマンド1 < file1 | コマンド2 //file1の内容を標準入力からコマンド1に取り込み、コマンド1の標準出力をコマンド２の標準入力に渡す
$ コマンド << 文字列 //指定した文字列が入力されるまで入力受付
$ コマンド1 ; コマンド2 //コマンド1の終了ステータスに関わらず、コマンド2を実行
$ コマンド1 && コマンド2 // コマンド1が実行できれば、コマンド2を実行
& コマンド1 || コマンド2 // コマンド1が実行できなければ、コマンド2を実行
